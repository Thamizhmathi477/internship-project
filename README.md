## 📌 Description
AI-powered disease prediction using Random Forest and Streamlit.

## 🚀 Features
- Symptom selection via checkboxes
- Instant diagnosis with confidence score
- Emergency symptom detection
- Downloadable report

## 🛠 Tech Stack
- Python, Streamlit, Scikit-learn, Pandas, NumPy

## ▶️ How to Run
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
